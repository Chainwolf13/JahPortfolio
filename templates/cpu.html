Gor-1000k

In my digital systems design class, I was tasked to create a 4-bit mips processor. To do this I utilized the Karnaugh Map pattern to derive boolean functions that would map 3 bits of instructions to their designated op codes. which would allow for addition,subtraction, logical functions such as AND, OR, SLT, and LI instructions.

On the right will display the block diagram that the entire code I wrote would be modeled by. I also included the output which shows byte level instructions executed at a clock frequency and the outputs that would be written to memory.
